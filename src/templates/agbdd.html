What is AGBDD?

AGBDD, which stands for Autonomous Generic Behavior Driven Development, is a newly defined development paradigm meant to help QA teams to become independent from development teams in automated testing.

It is an expansion of Behavior Driven Development (BDD), which increments to that paradigm the Generic Autonomous Testing model (GAT), which allows the creation of test cases that can be automated and executed without the need of direct test implementation.

This is achieved by the creation of a set of generic Statements which can be used to describe the general behavior of a type of system, and are implemented in an Open Source environment.
